Flight Manual

A simple CSS style for making things look like old school NASA flight manuals, because, honestly, what's cooler than old school NASA?

General Look and Feel

The look and feel of Flight Manual is inspired by Spotify's Apollo Website, and old NASA flight manuals.

Because I don't have much actual content right now, everything that follows is almost entirely filler text. At some point Flight Manual will be used to actually do something, but, unfortunately, that time is not right now.

Flight Manual is still being worked on, and nowhere near complete. If you're interested in using it, or following along with its development, you can find the source on GitHub.
